## 🚀 Premiers Pas

### 1. Votre Espace de Travail

Après connexion, vous arrivez sur votre **Dashboard Personnel** :

![Dashboard](./assets/teacher-dashboard.png)

**Zones importantes :**
- **📅 Planning** : Vue d'ensemble de votre progression annuelle
- **📚 Mes Contenus** : Accès rapide à vos dernières créations
- **👥 Mes Classes** : Tableau de bord par classe
- **💡 Suggestions IA** : Recommandations personnalisées

### 2. Configuration Initiale

#### Étape 1 : Profil Pédagogique
```
Paramètres > Mon Profil Pédagogique
```

Renseignez :
- Votre niveau d'enseignement principal
- Votre approche pédagogique préférée
- Vos contraintes (effectifs, matériel disponible)
- Vos objectifs pour l'année

```
Classes > Importer depuis Pronote/ENT
```

Ou création manuelle :
- Nom de la classe
- Effectif
- Niveau général
- Particularités (classe euro, SEGPA, etc.)

### 3. Navigation Rapide

**Raccourcis clavier :**
- `Ctrl/Cmd + N` : Nouveau contenu
- `Ctrl/Cmd + E` : Générer exercices
- `Ctrl/Cmd + K` : Recherche rapide
- `Ctrl/Cmd + /` : Aide contextuelle

2. **Sélectionnez le mode Assistant** :

## 📊 Évaluations et Corrections

### Création d'Évaluations

#### 1. Évaluation Formative Rapide

**Générateur Express :**
```
Type : QCM Auto-correctif
Durée : 15 minutes
Notions : Fractions (addition, simplification)
Questions : 10
Mode : Feedback immédiat
```

**Résultat** : Lien élève avec correction instantanée

#### 2. Évaluation Sommative Personnalisée

**Assistant Évaluation :**
1. Sélectionnez les compétences à évaluer
2. Définissez la répartition (Restitution/Application/Raisonnement)
3. Choisissez le barème (points ou compétences)
4. Générez 30 sujets uniques anti-triche

### Correction Assistée par IA

**Scan & Correction :**
1. **Scannez** les copies (app mobile ou scanner)
2. **L'IA détecte** :
   - Réponses correctes/incorrectes
   - Erreurs de raisonnement
   - Présentation et soin
3. **Validation enseignant** :
   - Ajustez les points si nécessaire
   - Ajoutez des commentaires
   - Générez le bulletin de notes

**Analyse des Erreurs :**
```
📊 Analyse Classe 5ème B - DS Fractions

Erreurs fréquentes :
1. Addition de fractions : 65% ne trouvent pas le dénominateur commun
2. Simplification : 40% oublient de simplifier le résultat
3. Problèmes : 30% erreur de modélisation

Recommandations :
- Remédiation sur dénominateur commun (30 min)
- Exercices de systématisation simplification
- Méthodologie résolution de problèmes
```

## ❓ FAQ Enseignants

### Questions Fréquentes

**Q : Puis-je modifier le contenu généré ?**
R : Oui, tout est 100% modifiable dans l'éditeur intégré.

**Q : Comment être sûr de la conformité programme ?**
R : Chaque contenu affiche un score de conformité. Les inspecteurs valident régulièrement notre base.

**Q : Et si je n'ai pas Internet en classe ?**
R : Mode hors ligne disponible. Synchronisation au retour de connexion.

**Q : Puis-je garder ma méthode personnelle ?**
R : MCG s'adapte à VOTRE pédagogie, pas l'inverse. Paramétrez vos préférences.

**Q : Les données élèves sont-elles protégées ?**
R : RGPD strict, hébergement France, anonymisation, suppression sur demande.

**Q : Quel gain de temps réel ?**
R : En moyenne 8-10h/semaine selon nos utilisateurs.

**Q : L'IA remplace-t-elle l'enseignant ?**
R : Non ! Elle libère du temps pour l'essentiel : la relation pédagogique.
